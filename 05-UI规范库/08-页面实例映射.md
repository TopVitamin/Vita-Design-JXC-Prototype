# 页面实例映射

## 一、文档目的

本文件用于把当前项目已经存在或已规划的页面，映射到具体规范条目，避免实现者或AI只看到抽象规则。

## 二、当前页面映射

### 2.1 工作台首页

对应规范：

- `01-系统级约束总则`
- `02-Token与代码基线`
- `03-页面模板与骨架规范`中的工作台模板
- `04-组件与交互规则`中的筛选、入口卡片、个人信息下拉
- `05-状态权限与异常规则`

重点约束：

- 首页是工作台，不是展示型首页
- 入口与模块区应服务高频操作
- 卡片感应克制

### 2.2 线索管理列表页

对应规范：

- `01-系统级约束总则`
- `02-Token与代码基线`
- `03-页面模板与骨架规范`中的列表页模板
- `04-组件与交互规则`中的筛选区、表格、分页、状态tag
- `05-状态权限与异常规则`

重点约束：

- 标题、筛选、操作、表格、分页分层明确
- 宽表格允许横向滚动
- 状态列、人员列、操作列必须统一风格

### 2.3 企业信息表单页

对应规范：

- `01-系统级约束总则`
- `02-Token与代码基线`
- `03-页面模板与骨架规范`中的新增编辑页模板
- `04-组件与交互规则`中的表单控件、提示信息块、可编辑子表格
- `05-状态权限与异常规则`

重点约束：

- 分组标题、说明区、双列表单、长文本、融资信息子表格、底部操作区顺序固定
- 子表格保持表格语义

### 2.4 顶栏个人信息下拉

对应规范：

- `01-系统级约束总则`
- `02-Token与代码基线`
- `04-组件与交互规则`中的个人信息下拉
- `05-状态权限与异常规则`

重点约束：

- 必须包含头像信息区、分组菜单、状态/进度/开关项、退出入口
- 不能简化成普通菜单列表

### 2.5 详情页示例

对应规范：

- `01-系统级约束总则`
- `03-页面模板与骨架规范`中的详情页模板
- `04-组件与交互规则`中的步骤条、信息块、明细区
- `05-状态权限与异常规则`

重点约束：

- 首屏优先呈现状态与主信息
- 明细区保持结构化信息

### 2.6 弹层表单示例

对应规范：

- `01-系统级约束总则`
- `03-页面模板与骨架规范`中的弹层表单模板
- `04-组件与交互规则`中的表单控件、可编辑子表格
- `05-状态权限与异常规则`

重点约束：

- 宽度走`520/720px`分档
- 标题区、表单区、底部操作区层级固定

## 三、后续新增页面怎么映射

新增页面时，至少补3件事：

1. 页面属于哪个模板
2. 页面依赖哪些组件规则
3. 页面有哪些状态和权限要求

如果这3件事仍无法归类，才允许补新规则。
